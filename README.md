# WP Image Poster

このプラグイン( **WP Image Poster** )は、画像アップロード時に自動で投稿を生成する軽量な WordPress プラグインです。JPEG 画像をアップロードすると、Exif データ（撮影日時・カメラモデル・絞り値・シャッタースピード・ISO・焦点距離・GPS 座標）を取得し、その情報を整形したドラフト投稿を作成します。
画像は投稿のアイキャッチ画像として設定され、GPS データがある場合は OpenStreetMap の埋め込み地図も付加されます。

---

## 主な機能
- **自動投稿作成** – JPEG をアップロードすると自動でドラフト投稿が作られます
- **Exif メタデータ抽出** – 撮影日時、カメラモデル、絞り値、シャッタースピード、ISO、焦点距離、GPS 座標をサポート
- **出力設定** – 設定画面で表示項目を選択可能
- **マップ表示** – GPS タグが存在すると OpenStreetMap の iframe を挿入
- **ローカライズ対応** – `src/languages/ja.yml` で日本語化済み。追加の YAML ファイルを作れば他の言語も簡単に追加可能
- **外部依存なし** – WordPress コア関数と PHP Exif 拡張機能のみ使用

---

## 要件
- WordPress 5.5 以降
- PHP 7.4+（Exif 拡張が有効）
- `build.sh` で ZIP を作成したい場合は `zip` コマンドが使えること

---

## インストール

1. プラグインソースをダウンロードし、`wp-content/plugins/wp-image-poster` フォルダー内に配置します。
   ```bash
   wp-content/plugins/wp-image-poster/
   ├── image-poster.php
   ├── build.sh
   ├── src/
   │   ├── languages/ja.yml
   │   └── …
   ```
2. WordPress 管理画面の【プラグイン】→【新規追加】より「WP Image Poster」を有効化します。

※ ZIP でのインストールなら `zip` を作って `wp-content/plugins/` にアップロードしてもOKです。

---

## 使い方

有効化後、**設定** メニューに **Exif to Post** が追加されます。

1. **設定 → Exif to Post** を開く。
2. 表示したい Exif フィールドにチェックを入れる／外す。
3. 「変更を保存」をクリック。

次に **メディア** → **新規追加** で JPEG 画像をアップロードします。アップロード完了と同時に、設定したフィールドの情報を表形式で投稿本文に挿入し、ドラフト投稿が生成されます。GPS 情報がある場合はマップ iframe も追加されます。

---

## 設定項目一覧

| フィールド | 説明 |
|---|---|
| **Date Taken** | EXIF タイムスタンプから生成される投稿日付 |
| **Camera Model** | カメラのメーカーとモデル |
| **Aperture (F-Number)** | レンズの絞り値 |
| **Shutter Speed** | 撮影速度（露光時間） |
| **ISO** | ISO 感度 |
| **Focal Length** | 焦点距離（mm） |
| **GPS Coordinates** | GPS 座標（リンク＋埋め込み地図） |

`src/languages/` 配下に `*.yml` ファイルを追加すると、各言語に合わせて自動ロードされます。

## トラブルシューティング

- **投稿が生成されない** – PHP の **Exif** 拡張が有効になっているか確認します。`<?php phpinfo(); ?>` を作成しブラウザで確認。
- **データが欠落** – 一部カメラは EXIF タグを別形式で保存するため、取得できない場合があります。
- **地図が表示されない** – GPS タグがないか、iframe URL が正しく生成されているか確認してください。
